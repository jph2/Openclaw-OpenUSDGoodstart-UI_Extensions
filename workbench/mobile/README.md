# Workbench Mobile

Mobile App für iOS/Android zum Anzeigen und Bearbeiten von Markdown-Dateien aus GitHub Repositories.

## Features
- GitHub Token Login
- Quick-Access zu Studio Framework & OpenClaw Workspace
- Markdown Viewer mit Syntax-Highlighting
- Inline-Editor für Dateien
- Direktes Committen zu GitHub

## Setup
```bash
npm install
npx expo start
```

## Deployment
```bash
npx eas build --platform ios    # iOS
npx eas build --platform android # Android
```
